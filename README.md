## Requirements
- Python v3+<br>

## Installation
```
git clone https://github.com/sumit-buddy/CovPy/

cd CovPy

pip3 install -r requirements.txt
```

## Usage
```
python3 CovPy.py
```
or
```
python3 CovPy-simple.py
```

## Features
- [x] Track status with country name or id.
- [x] Easy to use.
- [x] Using best covid sdk
